# DzResearch Connect

AI-powered platform to connect Algerian researchers based on semantic similarity of their work.

---

## Project Overview

DzResearch Connect is a web platform that helps researchers discover potential collaborators by analyzing research topics, publications, and affiliations.

Instead of simple keyword matching, the system uses **AI embeddings and similarity search** to recommend relevant researchers.

---

##  Problem

- Researchers in Algeria are fragmented
- Hard to find people working on similar topics
- No centralized intelligent search system

---

##  Solution

We built a system that:
- Collects research data (papers, authors)
- Transforms text into embeddings
- Computes similarity between researchers
- Recommends collaborators
- Provides a search interface
